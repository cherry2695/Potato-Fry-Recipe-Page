# 🥔 Potato Fry Recipe Page

A simple and responsive web page that displays the recipe for crispy potato fries. This project is built using **HTML** and **CSS** and is perfect for beginners learning basic web development and page styling.

---

## 📌 Features

- 📃 Clear recipe details: ingredients, preparation steps, and cooking tips
- 🎨 Clean and minimal design
- 📱 Responsive layout — works well on mobile and desktop
- 🧩 Simple HTML & CSS structure, easy to understand and extend

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
